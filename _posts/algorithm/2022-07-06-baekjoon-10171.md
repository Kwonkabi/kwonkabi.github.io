### 문제 링크

[**10171번: 고양이**](https://www.acmicpc.net/problem/10171)

<br>

### ❌ 나의 틀린 풀이

```js
console.log(`
\    /\
 )  ( ')
(  /  )
 \(__)|
`);
```

😇 템플릿 리터럴 줄바꿈 문제라고 생각했다.

<br>

### ⭕️ 나의 맞은 풀이

```js
console.log("\\    /\\");
console.log(" )  ( ')");
console.log("(  /  )");
console.log(" \\(__)|");
```

🥹 백슬래시를 출력하기 위한 이스케이프 시퀀스 활용 문제였다.

`"\\"` 이렇게 입력해야 => `"\"`이렇게 출력된다.

자주 사용되는 이스케이프 시퀀스로는 `\\`(백슬래시), `\n`(줄바꿈), `\'`(작은따옴표), `\"`(큰따옴표)가 있다.
